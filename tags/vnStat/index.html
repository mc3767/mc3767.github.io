CentOS / RHEL / Fedora에서 vnStat (네트워크 모니터링 도구)를 설정하는 방법

1, 2019
네트워크 인터페이스에서 대역폭 (전송 및 수신)을 계산하고 자체 데이터베이스에 저장하는 유틸리티입니다. 설치 및 설정 1. Yum을 사용하여 vnStat 패키지 설치 yum 명령 줄 유틸리티를 사용하여 시스템에 vnstat 패키지를 설치합니다. # yum install vnstat 2. 데이터베이스 초기화 vnStat는 모든 정보를 데이터베이스에 보관합니다. 아래 명령을 사용하여 네트워크 인터페이스 용 데이터베이스를 생성합니다. 이 기사는 em1 인터페이스를 사용하고 있습니다. 필요에 따라 인터페이스 이름을 변경하십시오. # vnstat -u -i eth0 3. 크론 예약 vnStat는 cron 작업을 사용하여 데이터베이스를 업데이트합니다.
