Post with a Background Image

26, 2013
Here be a sample post with a custom background image. To utilize this “feature” just add the following YAML to a post’s front matter. image: background: filename.png This little bit of YAML makes the assumption that your background image asset is in the /images folder. If you place it somewhere else or are hotlinking from the web, just include the full http(s):// URL. Either way you should have a background image that is tiled.

Syntax Highlighting Post

16, 2013
Syntax highlighting is a feature that displays source code, in different colors and fonts according to the category of terms. This feature facilitates writing in a structured language such as a programming language or a markup language as both structures and syntax errors are visually distinct. Highlighting does not affect the meaning of the text itself; it is intended only for human readers.1 GFM Code Blocks GitHub Flavored Markdown fenced code blocks are supported.

Sample Link Post

12, 2013
This theme supports link posts, made famous by John Gruber. To use, just add link: http://url-you-want-linked to the post’s YAML front matter and you’re done.
